# Algo8_assignment
Follow the steps below to run this project on your local machine:

###  Prerequisites
- Node.js installed
- MongoDB installed and added to system PATH
### 1. Clone the Repository
### 2. Install Dependencies
- Run "npm i" in terminal to install all dependencies
### 3. Start MongoDB server
- Start your MongoDB server by running "mongod" in seperate terminal
### 4. Run the application
- Run "node index.js" in the terminal.
### 5. Open in borwser
- the code will run in at localhost 3000. Go to "http://localhost:3000/user/signup" to sign up.
#### Note-  After signup and login a token will be generated which can be seen in cookies using inspect, the token is deleted after logout and delete operation
